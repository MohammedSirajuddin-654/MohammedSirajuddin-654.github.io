Post Graduate Diploma in Cloud Computing for Big Data

Lambton College

Sep 2020 - Apr 2022

At Lambton College, I pursued a Post Graduate Diploma in Cloud Computing for Big Data, a program that significantly enhanced my expertise in cloud technologies and big data analytics. This program was instrumental in shaping my understanding of how to design, implement, and manage robust cloud-based solutions.

Key Areas of Study:

  • Cloud Computing Fundamentals: Comprehensive understanding of cloud service models (IaaS, PaaS, SaaS) and deployment models (public, private, hybrid).
  • Major Cloud Platforms: Hands-on experience with leading cloud platforms, including Amazon Web Services (AWS) and Microsoft Azure.
  • Big Data Technologies: Exposure to big data tools and technologies, such as Hadoop, Spark, and NoSQL databases, for processing and analyzing large datasets.
  • Cloud Security: Best practices for securing cloud environments, including identity and access management, data encryption, and network security.
  • DevOps Practices: Implementing CI/CD pipelines, automation, and infrastructure as code using tools like Terraform, Jenkins, and Docker.
  • Cloud Architecture: Designing scalable, resilient, and cost-effective cloud architectures to meet various business needs.
  • Data Analytics: Leveraging cloud-based analytics services to derive insights from data.

Projects and Practical Experience:

  • Developed a cloud-based application using AWS services, including EC2, S3, Lambda, and RDS.
  • Implemented a CI/CD pipeline using Azure DevOps for automating the deployment of a web application.
  • Worked on a big data project using Hadoop and Spark to process and analyze large datasets.
  • Designed and implemented a secure cloud environment using Azure, incorporating best practices for identity and access management.

My education at Lambton College provided me with a strong foundation in cloud computing and big data, equipping me with the skills needed to tackle real-world challenges in the field. The program's focus on practical, hands-on learning was particularly valuable, allowing me to apply theoretical concepts to real-world scenarios.
